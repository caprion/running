# Daily Run Log

Quick post-run capture - fill this in right after runs if you want to track subjective feel.

## Format

```
DATE | DISTANCE | FEEL | NOTES
-----|----------|------|------
2026-01-07 | 6km easy | 7/10 | Felt sluggish first 2km, legs heavy from poor sleep
2026-01-09 | 8km tempo | 8/10 | Good session, hit paces, slight right knee awareness
2026-01-11 | 15km long | 6/10 | Struggled last 5km, too fast early, need better pacing
```

## Quick Reference

**Feel Scale:**
- 10/10: Amazing, could run forever
- 8/10: Strong, good workout
- 6/10: Average, got it done
- 4/10: Struggled, tough day
- 2/10: Survival mode

**What to Note:**
- Unexpected pains/niggles
- Mental state (motivated? dreading it?)
- External factors (heat, wind, hills, poor sleep)
- Pacing issues
- Form cues that worked/didn't work
